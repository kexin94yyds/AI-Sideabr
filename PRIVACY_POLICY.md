# AI Sidebar Privacy Policy

Last updated: March 26, 2026

## Overview

AI Sidebar is a Chrome extension that helps users open and use supported AI websites inside the browser side panel. The extension primarily operates locally in the user's browser.

## Data We Process

AI Sidebar may process the following categories of data in order to provide its features:

- Extension settings and UI preferences stored in Chrome local storage.
- Recent conversation links or navigation history needed for sidebar navigation features.
- Authentication state derived from browser cookies or identity flows when the user signs in to supported websites.
- Page URLs and page structure on supported websites so the extension can inject UI helpers and adjust layout behavior.

## How Data Is Used

This data is used only to:

- render the side panel and provider switching experience;
- remember user preferences;
- detect login state and supported pages;
- enable features such as layout adjustment, history shortcuts, export helpers, and related productivity tools.

## Data Sharing

AI Sidebar does not sell user data.

AI Sidebar does not transfer user data to the developer's own servers for core extension functionality.

When the user explicitly uses third-party AI websites or optional integrated services, data may be exchanged directly between the user's browser and those third-party services under their own terms and privacy policies.

## Remote Services

The extension loads and interacts with third-party websites selected by the user, such as AI service providers. Those websites may receive data directly from the user during normal use.

If optional productivity or sync-related features are enabled in the future, the related data flow should be disclosed in the extension listing and updated in this policy before release.

## Storage

Most extension data is stored locally using Chrome extension storage APIs. Authentication data used by supported websites remains subject to the browser and those websites' own authentication mechanisms.

## Permissions

The extension requests permissions such as `sidePanel`, `tabs`, `storage`, `cookies`, `identity`, `scripting`, `contextMenus`, and declarative network request permissions because those are required to render the side panel, detect sign-in state, inject page helpers, and improve compatibility with supported sites.

## User Controls

Users can:

- remove the extension at any time through Chrome's extension manager;
- clear local extension data through Chrome browser settings;
- stop using any supported third-party website at any time.

## Children's Privacy

AI Sidebar is not directed to children under 13.

## Changes

This privacy policy may be updated when the extension's features or data practices change. Material changes should be reflected in the Chrome Web Store listing before release.

## Contact

For privacy questions, provide a support contact in the Chrome Web Store listing before publishing.
